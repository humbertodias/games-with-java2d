# Games with Java 2D

Simple Games using Java 2D


## Prerequires

1. Git 2.6+
2. Maven 3+
3. Java 8+


## How to Play

Clone

```
git clone https://github.com/humbertodias/games-with-java2d.git
```

Inside the folder

```
cd games-with-java2d
```

Run

```
mvn compile exec:java -Dexec.mainClass="pacman.Main"
```
